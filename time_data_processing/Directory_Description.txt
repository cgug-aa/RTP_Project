차량 이동 경로 데이터의 시간 데이터를 사용하여 사용자의 이동 시간을 전처리하는 작업.

1st) 기존 이동 데이터 csv 파일에 start, end, duration 타임스템프의 시간(hour)을 기록하여 칼럼으로 추가한다.
2nd) 하루의 시간을 3구간 or 4구간으로 분리하여, 특정 id(한사람)의 이동 시간 데이터들이 어느 범주에 해당하는지 카운팅한다.
    Ex) 3구간 
                6~11시
                12시~18시
                19시~5시
        4구간
                5~10시
                11~15시
                16시~19시
                20시~4시
